# docs

General documentation for the Whisperwood Universe repository.

## Purpose

This folder holds supporting documentation that is not canonical story content — such as contributor guides, process notes, glossaries, and reference material for humans and AI assistants.

## For Humans

Use this folder for how-to guides, editorial workflows, and repository documentation that does not belong in [CANON.md](../CANON.md).

## For AI Assistants

Do not treat files here as canon unless they explicitly reference or restate [CANON.md](../CANON.md). Prefer placing world facts in [lore/](../lore/), [characters/](../characters/), and related content folders. Read [PROJECT.md](../PROJECT.md) for documentation standards.
