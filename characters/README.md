# characters

Character profiles for the Whisperwood Universe.

## Purpose

This folder holds canonical records for characters — names, personalities, relationships, and approved traits. Content may use Markdown (`.md`) for narrative profiles and JSON (`.json`) for structured metadata.

## For Humans

Add one record per character. Keep personalities consistent with [CANON.md](../CANON.md). Character changes require intentional canon updates and explicit approval.

## For AI Assistants

Do not invent characters or alter established personalities without approval. Each character should have strengths and weaknesses; no character is perfect. Cross-reference [schemas/](../schemas/) when structured JSON is used.
