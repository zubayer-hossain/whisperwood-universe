# references

External and mood reference material for this character.

---

## Purpose

Visual references that guide design without being canonical artwork — mood boards, real-world inspiration, palette swatches, and style comparisons.

---

## What Belongs Here

- Mood boards (collage images)
- Color palette references
- Style references ("drawn like this, but…")
- Fabric, texture, or material swatches
- Licensed reference with documented usage rights

---

## What Does Not Belong Here

- Final character illustrations → [images/](../images/)
- Original concept sketches → [concepts/](../concepts/)

---

## Guidelines

- Document sources when using external reference (filename or README note)
- Do not use copyrighted reference without permission
- Reference files are for production; they are not published as canon art

---

## Linking

Optional reference in `profile.json` → `media` with `type`: `other` and descriptive `label`.
