# prompts

AI prompts for Whisperwood Universe creative and editorial tasks.

## Purpose

This folder holds reusable prompt templates for AI-assisted writing, review, consistency checks, and metadata generation — always constrained by [CANON.md](../CANON.md).

## For Humans

Write prompts that instruct AI tools to preserve canon, respect content boundaries, and avoid inventing unapproved material.

## For AI Assistants

When executing prompts from this folder, still treat [CANON.md](../CANON.md) as the highest authority. Prompts guide process; they do not override canon or character rules.
