# assets

Reference assets for the Whisperwood Universe.

## Purpose

This folder holds non-code reference materials — such as illustrations, logos, color palettes, and design references — that support storytelling and brand consistency.

## For Humans

Store approved visual and brand assets here. Large binary files should be managed thoughtfully; document usage rights and sources where applicable.

## For AI Assistants

Do not generate or place story text canon in this folder. Assets are reference material only. Respect [style-guides/](../style-guides/) and [CANON.md](../CANON.md) when describing or referencing visual direction.
