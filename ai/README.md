# ai

AI-specific resources for the Whisperwood Universe.

## Purpose

This folder holds materials that help AI assistants work effectively in this repository — such as context packs, index files, workflow notes, and assistant-oriented summaries that point to authoritative sources.

## For Humans

Use this folder to reduce repetition for AI tools while keeping [CANON.md](../CANON.md), [PROJECT.md](../PROJECT.md), and [AGENTS.md](../AGENTS.md) as the primary authorities.

## For AI Assistants

Files here are navigational and operational aids. They do not override [CANON.md](../CANON.md). Do not store invented lore in this folder. When in doubt, read canon and content folders directly.
