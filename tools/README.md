# tools

Utility tooling for the Whisperwood Universe repository.

## Purpose

This folder is reserved for scripts and utilities that support validation, formatting, or maintenance of repository content — for example, schema validation or consistency checks.

## For Humans

Add tooling here when the repository needs automated support. Document each tool clearly. This repository remains a canon source of truth, not an application.

## For AI Assistants

Do not add implementation code unless explicitly requested. Tools must not modify canon content without human approval. Story and world facts belong in content folders, not here.
