# style-guides

Style and voice guidelines for the Whisperwood Universe.

## Purpose

This folder holds editorial and creative standards — narrative voice, tone, naming conventions, visual direction notes, and format guidelines for books, animation, and other media.

## For Humans

Use this folder to keep Whisperwood feeling cohesive across contributors and formats. Align all guides with [CANON.md](../CANON.md) themes and storytelling principles.

## For AI Assistants

Apply style guides when generating or editing content. Guides describe *how* to write and present material; [CANON.md](../CANON.md) defines *what* the universe is. Do not introduce tone or content that conflicts with canon.
