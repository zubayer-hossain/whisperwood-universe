# .cursor/rules

Cursor IDE rules for working in the Whisperwood Universe repository.

## Purpose

This folder contains rule files that guide AI assistants in Cursor. Rules reinforce editorial standards, canon authority, and safe editing practices.

## For Humans

Add or update `.mdc` rule files here to give Cursor persistent context about how this repository should be edited.

## For AI Assistants

Rules in this folder supplement [AGENTS.md](../../AGENTS.md), [PROJECT.md](../../PROJECT.md), and [CANON.md](../../CANON.md). Follow them when present. Never use this folder for story content or structured canon data.
