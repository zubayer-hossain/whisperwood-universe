# Whisperwood Universe

The official source of truth for **Whisperwood** — an original children's storytelling universe and home of **Zulk & Zaya**.

This repository holds canonical documentation and structured data for the universe. It is **not** an application: there is no application code, build tooling, or runtime services here.

---

## About

**Whisperwood** is a warm, imaginative world built for young listeners and early readers. The flagship series, **Zulk & Zaya**, follows magical adventures rooted in kindness, curiosity, and friendship.

Content in this universe is written for **children aged 4–8**.

---

## What This Repository Contains

- Universe and series overview
- Character profiles and relationships
- Locations and world geography
- Story outlines, episodes, and narrative arcs
- Editorial guidelines and glossary terms
- Structured metadata for humans and AI tools

Documentation uses **Markdown** (`.md`). Structured data uses **JSON** (`.json`).

## Core Themes

Stories and canon material in Whisperwood reflect:

- **Kindness** — treating others with care and compassion
- **Curiosity** — wondering, asking questions, and learning
- **Friendship** — building and keeping meaningful connections
- **Adventure** — exploring the world with courage and joy
- **Hope** — believing things can get better
- **Imagination** — dreaming, creating, and seeing possibilities

## Contributing

Anyone editing this repository — human or AI — should:

1. Read [README.md](README.md), [PROJECT.md](PROJECT.md), and [CANON.md](CANON.md) first
2. Preserve consistency with existing canon and project rules
3. Avoid inventing lore or changing established characters without explicit approval
4. Keep all material appropriate for ages 4–8

Whisperwood material must never include horror, gore, violence, cynicism, or adult humor.

For full contributor responsibilities and content boundaries, see [PROJECT.md](PROJECT.md).
