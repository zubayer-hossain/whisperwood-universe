# lore

World lore and background detail for the Whisperwood Universe.

## Purpose

This folder holds canonical background material — history, customs, natural laws, and world-building detail that supports stories without overriding [CANON.md](../CANON.md).

## For Humans

Use this folder for depth that writers and creators need, organized clearly and kept consistent over time.

## For AI Assistants

[Lore](../lore/) expands on canon; it does not replace it. Never invent lore unless explicitly requested and approved. Align all material with world rules and content boundaries in [CANON.md](../CANON.md).
