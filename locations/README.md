# locations

Locations and geography of the Whisperwood Universe.

## Purpose

This folder holds canonical records for places — regions, landmarks, homes, and settings where stories take place. Content may use Markdown (`.md`) for descriptions and JSON (`.json`) for structured metadata.

## For Humans

Document locations clearly enough for writers, illustrators, and future media teams to stay consistent across stories.

## For AI Assistants

Do not invent locations unless explicitly requested and approved. Respect world rules in [CANON.md](../CANON.md) — including gentle magic, child safety, and respect for nature. Cross-reference [schemas/](../schemas/) when structured JSON is used.
