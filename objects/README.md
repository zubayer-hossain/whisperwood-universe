# objects

Significant objects and items in the Whisperwood Universe.

## Purpose

This folder holds canonical records for important objects — tools, keepsakes, magical items, and story-relevant artifacts. Content may use Markdown (`.md`) for descriptions and JSON (`.json`) for structured metadata.

## For Humans

Document only objects that matter to continuity or storytelling. Keep descriptions appropriate for ages 4–8.

## For AI Assistants

Magic in Whisperwood is gentle and serves the story; objects should reflect that. Do not invent items unless explicitly requested and approved. Cross-reference [schemas/](../schemas/) when structured JSON is used.
