# schemas

JSON schemas for structured Whisperwood Universe data.

## Purpose

This folder holds JSON Schema (or equivalent) definitions for characters, locations, objects, episodes, and other structured records used across the repository.

## For Humans

Define schemas here before adding large amounts of structured JSON in content folders. Keep schemas aligned with [PROJECT.md](../PROJECT.md) documentation standards.

## For AI Assistants

Validate structured data against schemas in this folder when present. Schemas describe data shape only — not story content. Canonical facts still come from [CANON.md](../CANON.md) and approved records in content folders.
