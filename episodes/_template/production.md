# Production Checklist — Example Episode Title

> **Episode ID:** `episode-example`  
> **Status:** Template — track progress per stage

Update [episode.json](episode.json) → `productionStatus` when major stages complete.

---

## Purpose

This checklist tracks the episode from **outline through publishing**. Use it for human production teams and as a reference for Story Studio pipeline stages.

Mark each item: `[ ]` not started · `[~]` in progress · `[x]` complete · `[-]` not applicable

## Production Status Reference

Suggested `productionStatus.stage` values for [episode.json](episode.json):

| Stage | Meaning |
|-------|---------|
| `outline` | Beats and JSON draft |
| `writing` | episode.md and storyboard in progress |
| `review` | Editorial review |
| `pre-production` | Approved; illustration/voice prep |
| `production` | Active asset creation |
| `post-production` | Edit, mix, finalize |
| `published` | Released |
| `archived` | Retired from active catalog |
