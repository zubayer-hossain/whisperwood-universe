# episodes

Story episodes and narrative arcs for the Whisperwood Universe.

## Purpose

This folder holds outlines, scripts, and episode-level documentation for **Zulk & Zaya** and other approved series. Content may use Markdown (`.md`) for story notes and JSON (`.json`) for structured metadata.

## For Humans

Each episode should follow storytelling principles in [CANON.md](../CANON.md): curiosity, kindness, teamwork, hope, and teaching without preaching.

## For AI Assistants

Do not write full story content unless requested. Do not include horror, gore, cruelty, bullying as comedy, adult humor, politics, religion, or real-world conflict. End stories with hope. Cross-reference [schemas/](../schemas/) when structured JSON is used.
